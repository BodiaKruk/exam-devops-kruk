# task01: Terraform DigitalOcean

## Опис
- Розгортання VPC, Firewall, Droplet, Spaces bucket
- tfstate зберігається у DigitalOcean Spaces

## Як використовувати
- Заповніть secrets у GitHub: DO_TOKEN, SPACES_KEY, SPACES_SECRET, SSH_FINGERPRINT
- Для dry-run: terraform plan
- Для реального розгортання: terraform apply
- Для знищення: terraform destroy
